## Chart/Graphic Technical Specs

### Color Palette for Data

```css
/* Primary data colors */
--data-primary: #D4A574;     /* Gold - your metrics */
--data-secondary: #3B82F6;   /* Blue - industry/comparison */
--data-positive: #22C55E;    /* Green - growth/good */
--data-negative: #EF4444;    /* Red - problems/leaks */
--data-neutral: #94A3B8;     /* Gray - background bars */

/* Chart backgrounds */
--chart-bg: #0F172A;
--chart-grid: rgba(148, 163, 184, 0.1);
--chart-border: rgba(148, 163, 184, 0.2);
```

### Typography in Charts

- **Large numbers:** Inter Bold, 48-72px
- **Labels:** Inter Medium, 14-16px
- **Captions:** Inter Regular, 12-14px
- **All text:** White or --text-secondary (#E2E8F0)

### Animation Timing

- **Number count-up:** 1.5s ease-out
- **Bar grow:** 1s ease-out, stagger 0.2s between bars
- **Fade in:** 0.6s ease-out
- **Hover effects:** 0.2s ease
