## Dependencies & Critical Path

### Critical Path Items
1. System access (Day 1) — blocks everything
2. Audit completion (Week 3) — blocks process work
3. Setter hired (Week 7) — blocks team operations
4. Closer hired (Week 8) — blocks sales capacity
5. Reactivation launch (Week 9) — blocks major revenue opportunity

### External Dependencies
- Client availability for calls and approvals
- Candidate availability for hiring
- GHL platform stability
- Data quality for segmentation

---

## Risk Factors & Mitigation

| Risk | Impact | Mitigation |
|------|--------|------------|
| Delayed access | Week+ delay | Confirm all access before Day 1 |
| Hiring takes longer | Phase 2 extends | Start recruiting Week 4 |
| Data quality issues | Segmentation problems | Audit data quality in Week 1 |
| Stakeholder availability | Approval delays | Schedule all reviews in advance |
| Show rate doesn't improve | ROI question | Test multiple sequence variations |
